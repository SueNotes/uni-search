# 🎓 전국 대학 학과 정보 검색 서비스

## 1. 프로젝트 소개
이 프로젝트는 교육부의 공공데이터인 **'전국대학별학과정보'**를 활용하여, 수험생과 진로 지도 교사들이 전국의 대학 및 학과 정보를 손쉽게 검색할 수 있도록 만든 **웹 기반 검색 서비스**입니다.

기존의 복잡한 엑셀 파일을 열어볼 필요 없이, 웹 브라우저에서 지역별, 유형별, 키워드별로 원하는 정보를 즉시 찾을 수 있습니다.

### 🌟 주요 기능
- **지역별 검색**: 서울, 경기, 부산 등 원하는 시도의 대학만 필터링
- **학교 유형 필터**: 4년제 대학교, 2년제 전문대학, 대학원 구분 검색
- **통합 키워드 검색**: 학교 이름뿐만 아니라 **개설 학과**, **관련 직업** 키워드로도 검색 가능
- **직관적인 결과 카드**: 학교별로 개설된 학과 목록을 모아서 한눈에 제시
- **홈페이지 연결**: 검색된 대학의 공식 홈페이지로 바로 이동 가능

---

## 2. 기술적 구현 과정 (Data Pipeline)

이 서비스는 **데이터 분석(Python)**과 **웹 개발(HTML/JS)** 기술이 융합된 결과물입니다.

### STEP 1. 데이터 수집 (Raw Data)
*   **출처**: 공공데이터포털 / 커리어넷 오픈 API
*   **형식**: 대용량 엑셀 파일 (`.xls`)
*   **한계**: 용량이 크고(수십 MB), 웹 브라우저에서 직접 읽기에는 무거움.

### STEP 2. 데이터 가공 (Preprocessing)
*   **사용 언어**: Python (Pandas 라이브러리)
*   **변환 작업 (`excel_to_json.py`)**:
    1.  엑셀 데이터를 불러와 불필요한 정보 제거
    2.  웹에서 사용하기 쉽도록 한글 컬럼명을 영문으로 변경
    3.  **JSON 포맷**으로 변환하여 저장
*   **결과물**: `university_data.json` (가볍고 빠른 텍스트 데이터)

### STEP 3. 웹 서비스 개발 (Frontend)
*   **사용 언어**: HTML5, CSS3, JavaScript (Vanilla JS)
*   **구현 방식 (`university_web_app.html`)**:
    *   `fetch()` 함수를 사용하여 JSON 데이터를 비동기로 로딩
    *   사용자의 선택 조건(지역, 유형, 키워드)에 따라 실시간 필터링
    *   중복된 학교 데이터를 하나로 묶어(Grouping) 보여주는 알고리즘 적용

---

## 3. 사용 방법 (How to use)

1.  **데이터 로딩 확인**: 페이지 접속 시 "데이터 준비 완료!" 메시지를 확인합니다. (최초 1회 데이터 다운로드)
2.  **조건 선택**:
    *   **지역 선택**: 검색하고 싶은 시/도를 선택합니다. (예: 서울)
    *   **유형 선택**: 4년제, 2년제 등 학교 유형을 선택합니다.
3.  **검색어 입력**: 찾고 싶은 **대학 이름**, **학과명**(예: 간호, 컴퓨터), 또는 **관련 직업**(예: 프로그래머)을 입력합니다.
4.  **검색하기**: 버튼을 누르면 조건에 맞는 모든 대학 목록이 카드 형태로 표시됩니다.

## 4. 설치 및 실행 가이드

이 프로젝트는 별도의 서버 설치 없이 브라우저만 있으면 실행 가능합니다.

### 로컬에서 실행하기
1.  이 저장소(Repository)를 다운로드합니다.
2.  `university_data.json` 파일이 `university_web_app.html`과 같은 폴더에 있는지 확인합니다.
3.  보안 정책상 HTML 파일을 직접 더블클릭하면 데이터 로딩이 차단될 수 있습니다.
    *   **VS Code 사용 시**: 'Live Server' 확장 프로그램을 사용하여 실행 (우클릭 -> Open with Live Server)
    *   **Python 사용 시**: 터미널에서 `python -m http.server` 입력 후 `localhost:8000` 접속

### 웹 배포하기 (Github Pages)
1.  Github 저장소에 `html` 파일과 `json` 파일을 업로드합니다.
2.  [Settings] -> [Pages] 메뉴에서 배포할 브랜치(main)를 선택합니다.
3.  생성된 URL을 공유합니다.

---

> **Tip**: 데이터를 업데이트하고 싶다면 최신 엑셀 파일을 다운로드 받아 파이썬 스크립트(`excel_to_json.py`)만 다시 실행해주면 됩니다!
